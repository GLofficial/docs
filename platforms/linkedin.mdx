| | |
|---|---|
| **Caption limit** | 3,000 characters |
| **Post types** | Text, image, video, carousel, document (PDF), GIF |
| **Capabilities** | Publish · Comments |
| **Connect** | Headless (`POST /v1/social/connect/linkedin`) or dashboard |
| **Tokens** | ~60 days |

Both **personal profiles and organization pages** are supported. Some analytics require organization-admin scope; if it's missing the account still connects and posts normally.

See [all platforms](/platforms) for the capability matrix and media rules.
