| | |
|---|---|
| **Caption limit** | 800 characters (description) |
| **Post types** | Pins, video pins |
| **Capabilities** | Publish · Comments (fetch + reply only) |
| **Connect** | Headless (`POST /v1/social/connect/pinterest`) or dashboard |
| **Tokens** | ~30 days |

Pinterest comments **cannot be hidden or deleted** via the API — those operations return `not_supported`.

See [all platforms](/platforms) for the capability matrix and media rules.
