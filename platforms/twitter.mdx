| | |
|---|---|
| **Caption limit** | 280 characters |
| **Post types** | Text, image, video |
| **Capabilities** | Publish · Comments · Direct messages |
| **Connect** | Headless (`POST /v1/social/connect/twitter`) or dashboard |
| **Tokens** | ~2 hours, auto-refreshed — reconnect if an account turns unhealthy |

No analytics via the API. Comments (replies) support full fetch / reply / hide / delete.

See [all platforms](/platforms) for the capability matrix and media rules.
