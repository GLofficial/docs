| | |
|---|---|
| **Caption limit** | Not length-checked (Stories carry no caption) |
| **Post types** | Stories, Spotlight |
| **Capabilities** | Publish |
| **Connect** | Headless (`POST /v1/social/connect/snapchat`) or dashboard |
| **Tokens** | ~1 hour, auto-refreshed |

Requires a Snapchat Public Profile. No comments or DM inbox.

See [all platforms](/platforms) for the capability matrix and media rules.
