| | |
|---|---|
| **Caption limit** | 2,200 characters |
| **Post types** | Feed, Carousel, Reels, Stories |
| **Capabilities** | Publish · Analytics · Comments · Direct messages |
| **Connect** | Dashboard |
| **Tokens** | ~60 days, auto-refreshed |

**Requires media** — a text-only post is rejected. DMs use Meta's **24-hour messaging window**.

See [all platforms](/platforms) for the capability matrix and media rules.
